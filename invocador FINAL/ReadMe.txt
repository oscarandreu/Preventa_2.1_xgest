========================================================================
    APLICACIÓN WIN32: Información general del proyecto invok
========================================================================

El Asistente para aplicaciones ha creado esta aplicación de invok.  
Archivo que contiene un resumen del contenido de los archivos que
componen la aplicación de invok.


invok.vcproj
    Archivo de proyecto principal para proyectos de VC++ generados con el Asistente para aplicaciones. 
    Contiene información acerca de la versión de Visual C++ que generó el archivo y 
    de las plataformas, configuraciones y características del proyecto
    seleccionadas con el Asistente para aplicaciones.

invok.cpp
    Éste es el archivo de código fuente principal de la aplicación.

/////////////////////////////////////////////////////////////////////////////
El Asistente para aplicaciones ha creado los siguientes recursos:

invok.rc
    Listado de los recursos de Microsoft Windows
    que utiliza el programa. Incluye iconos, mapas de bits y cursores almacenados
    en el subdirectorio RES. Este archivo se puede editar directamente en
    Microsoft Visual C++.

Resource.h
    Archivo de encabezado estándar que define nuevos identificadores de recursos.
    Microsoft Visual C++ lee y actualiza este archivo.
invok.ico
    Archivo de icono que se utiliza como icono de la aplicación (32x32).
    Este icono está incluido en el archivo de recursos principal invok.rc.

small.ico
    Archivo de icono que contiene una versión más reducida (16x16)
    del icono de la aplicación. Este icono está incluido en el
    archivo de recursos principal invok.rc.

/////////////////////////////////////////////////////////////////////////////
Otros archivos estándar:

StdAfx.h, StdAfx.cpp
    Archivos utilizados para generar un archivo de encabezado precompilado (PCH)
    llamado invok.pch, así como un archivo de tipos precompilado llamado StdAfx.obj.

Notas adicionales:

se deben agregar o personalizar.
