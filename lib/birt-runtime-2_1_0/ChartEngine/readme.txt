				               BIRT Chart Runtime
				       
 What is it? 
 -----------  
  BIRT is an open source, Eclipse-based reporting system that integrates with 
your application to produce compelling reports for both web and PDF.   
  BIRT Chart is a 2 or 2.5 dimension data representation component.It can produce
24-bit images,8-bit images,SWT graphics or SWING graphics.BIRT Chart Runtime includes
the resource which user can use to integrate Chart capability into its own 
application.

 The Website for BIRT.
 ---------------------
  You can find the manual,samples and other source from http://www.eclipse.org/birt.

 
 Running the BIRT Chart Examples
 ---------------------------------- 
  In order to run the BIRT Chart Examples you should add the SWT JAR file to the Java build path of the project.
		
  For STANDLONE application, set the VM arguments as follows:
	-DSTANDLONE
	-DBIRT_HOME=<BIRT_Home>
 
 More information about BIRT Chart Examples could be found at readme.html within org.eclipse.birt.chart.examples plugin.
